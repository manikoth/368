Web Design 1

P2: Bio

Objectives

  • Introduce yourself, experience, & education to potential employers/collaborators
  • Combine a bio and also resume into a single page
  • Utilize web medium to deliver static content, i.e. HTML vs. PDF

Requirements

Content

  1. Go to the backend of your repo, e.g. github.com/user
  2. Create a wik page named bio
  3. Using Markdown syntax, create/copy the folowing content from your resume:
    • Introduce yourself, major, & professional objectives
    • Describe your education
    • Describe your Work/Volunteer/Activity/etc. experiences
    • Describe your skillset(s)
    • Optionally, add other relevant information

Design

  1. Clone your user.github.io repo
  2. Create page named bio.html & save to folder root, **user.github.io
  3. Convert Markdown content from bio wiki, to HTML.
  4. Markup with HTML elements to create structure.
  5. Style with CSS properties to create design.
  6. Reference How to Make a Website and also CSS Basics

Submission

  • On Canvas, submit URL the bio.html page front-end of your user.github.io repo, e.g. user.github.io/bio.html
