Web Design 1

P3: Style Tile

Moodboard too vague. Mockup too specfic. Style Tile, just right.

Objectives

  • Determine visual elements for your website
  • Create a Style Tile
  • Apply the research within the visual design process

Overview

Websites are so much more than just usable interfaces: they tell a story

  • Style Tiles by Samantha Warren
  • A List Apart - Style Tiles and How They Work
  • Style Tiles in Use

Content

  1. Go to the backend of your user.github.io repo, e.g. github.com/user
  2. Create a wiki page named style-tile
  3. Using Markdown syntax, address the following:
    • How do you want you and your work to be perceived?
    • What visual elements will work? What will not work?
    • What kind of content will you have: e.g. code samples, logos, essays, etc.

Design

Determine the visual elements that will exist on your site. For each, how will you use: Line, Form, & Space?

  • Typography
    • TH - Choosing Text Typeface
    • TH - Choosing Display Typeface
    • REF: Google Fonts or CSS Font Stack
  • Colors
    • REF: TH - Color Theory
  • Patterns / Textures
    • REF: TH - Color, Texture, Value

Code

  1. Create new or add-to exsiting style-tile.html to your user.github.io repo
  2. Copy & paste the following code for:
    • style-tile.html
    • add to styles.css

Submission

  • On Canvas, submit URL the backend of your user.github.io repo, e.g. github.com/user
