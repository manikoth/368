Web Design 1

P2: Resume

Objectives

  • Easily updated one page online resume
  • Apply Web Typography
  • Apply Box Model, Specificity, Advanced CSS Selectors properties to TreeHouse Grid System

Create Content

Create content in markdown format.

  1. Setup new repository named p2-resume
  2. Create a Wiki page for this p2-resume repo
  3. Provide URL in submission form - see below

C. Markdown to HTML

Convert Markdown to HTML using CodePen.

  1. Copy & paste content from your Wiki page
  2. Convert markdown to HTML
  3. Copy converted HTML

Get Files

Using our Website Starter create a one-page resume.

  1. Download files from Website Starter
  2. Add the contents of Website Starter to your p2-resume
  3. Make first commit, and create gh-pages branch. Again, Do work on gh-pages, not master branch.

Do Work

Work local. Design. Iterate. Repeat.

  • Work locally
  • Demonstrate design principles
  • Use Google font
  • Use TreeHouse grid system

Go Live

Stage local website remotely on GitHub.

  1. Push local commits live
  2. Synch gh-pages branch
  3. Website should be accesbile at: YourUserName.github.io/p2-resume
